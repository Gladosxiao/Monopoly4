## 视频 1：转转盘机制（保险公司、旅馆、百货商店、机场）

**来源**：`[首发]大富翁4顶尖高手教你转转盘(保险公司、旅馆、百货商店、机场).mp4`

**关键帧数量**：90 帧（3 页 contact sheet）

**弹幕数量**：0 条

### 观察要点

- 视频主要演示进入 4 类公司地块后触发的**轮盘（转盘）机制**。
- 轮盘结果决定：停留天数、消费/保费金额、出国天数、获得的卡片/道具等。
- 保险公司：强制签保，轮盘决定投保天数并支付保费；投保期间住院可获得理赔。
- 旅馆：轮盘决定休息天数及住宿费；住宿期间无法行动、不能收租。
- 百货商店：轮盘决定可购买的商品或折扣，可用点券购买卡片/道具。
- 机场：轮盘决定出国观光天数；出国期间无法行动、不能收租。
- 公司地块效果与 `CompanyType` 一一对应，已在 `docs/design/04-game-rules.md` 中记录。

### 设计补充

- 公司地块触发轮盘时，应显示一个带权重/随机区间的轮盘 UI。
- 轮盘结果应写入 `GameLog`，并更新玩家 `statusEffects`（如 `insurance`、`abroad`、`hotelRest` 等）。

## 视频 2：神明、道具、卡片妙用

**来源**：`【大富翁4】神明、道具、卡片有哪些妙用？土地爷是真的狗！那个不可描述之神是什么东西？【妙棋物语18.1】.mp4`

**关键帧数量**：637 帧（16 页 contact sheet）

**弹幕数量**：247 条

### 一、神明系统细节

| 神明 | 观察到/确认的效果 | 备注 |
|---|---|---|
| 土地公公 | 附身后走到哪里**免费强占**到哪（包括对手土地）；7 天后消失，隔月才重新出现 | 可配合乌龟卡缓慢推进，连续占地 |
| 小财神 | 获得小额现金；过路费减半 | 7 天后可变大财神 |
| 大财神 | 获得大额现金；**免除过路费**；部分命运罚金可挡掉 | 7 天后可变回小财神 |
| 小福神 | 获赠随机卡片；买地半价；免费加盖一层 | 7 天后可变大福神 |
| 大福神 | 获赠 2 张卡片；买地免费；免费加盖 | 7 天后可变回小福神 |
| 小穷神 | 向其他玩家支付小额现金；踩到别人土地时租金 +50% | 7 天后可变大穷神 |
| 大穷神 | 损失大额现金；过路费翻倍 | 7 天后可变回小穷神 |
| 小衰神 | 遗失 1 张卡片；买地/投资失败 | 7 天后可变大衰神；可能把送神符弄丢 |
| 大衰神 | 遗失一半卡片（奇数时减一除以二）；买地/投资失败；罚金翻倍 | 弹幕确认：只有 1 张卡时反而不会丢 |
| 天使 | 经过土地免费加盖一层（不分敌我） | 7 天后变恶魔 |
| 恶魔 | 经过土地拆除一层（不分敌我） | 7 天后变天使 |
| 死神 | 认输投降时由魔法屋召唤；附身 13 天（部分弹幕说 14 天）；损失全部卡片道具；所有损失转嫁其身；送神符无效 | 多人模式专属 |
| 不可描述之神 | 视频中玩笑式称呼，可能指隐藏/特殊神明效果，实际游戏中通常对应大衰神或恶魔 | 未观察到独立于 12 主神之外的新神明 |

**重要规则确认**：
- 同一时间只能有一种神明附身，新神明会替换旧神明。
- 送神符对**好神无效**，只能送走坏神（衰、穷、恶魔、死神），也可送走身上的定时炸弹。
- 土地公公 7 天后直接消失，不变身为其他神明。
- 恶犬属于地图 NPC/动物，**不属于神明系统**。

### 三、卡片系统细节

- 卡片需要消耗**点券**购买，不同卡片价格（点数）不同：观察到 10、15、20、25、30、35、40、50、70、100、160、180、200 点等价位。
- 持有上限：每人最多 15 张。
- 额满后再获得需选择舍弃一张。

**常用卡片与组合技**（来自关键帧与弹幕）：

| 卡片 | 效果 | 组合/技巧 |
|---|---|---|
| 乌龟卡 | 令目标每次只走一步，持续 3 天 | 配合土地公公连续占地；配合路障/地雷制造连招 |
| 停留卡 | 令目标下次移动时原地停留一回合 | 配合百货商店反复购买道具卡片 |
| 转向卡 | 改变目标下一次行走方向 | 配合乌龟卡、路障使用 |
| 拆除卡 | 拆建筑一级，或清除地雷/路障/定时炸弹 | 可拆炸弹；配合土地公等于挖掘机 |
| 嫁祸卡 | 将罚金/损失转嫁给指定对手 | 到对手土地付路费后嫁祸回去 |
| 免费卡 | 免除一次房租/罚款/税金 | 配合查封卡可形成防御链 |
| 梦游卡 | 目标梦游 5 天，随机行走，不能买地/收租/用卡 | 命运/新闻不触发；配合抢夺卡可抢车 |
| 抢夺卡 | 随机抢目标 1 张卡片或 1 个道具 | 可抢车、抢卡 |
| 冬眠卡 | 所有对手冬眠 5 天，无法行动和收租 | 可配合其他控制卡 |
| 复仇卡 | 被陷害/入狱/催眠时自动反击施加者 | 身上有复仇卡时倒闭可触发死神 |
| 送神符 | 送走坏神或定时炸弹 | 衰神可能让你先把送神符丢了 |
| 请神符 | 招来最近的神明附身 | 争取好神/转移坏神 |
| 红卡/黑卡 | 指定股票连涨/连跌 3 天 | 炒股核心卡；红卡最强 |

### 四、特殊人物与机制

- **四大恶人**（小偷、强盗、流氓、间谍）：保释后成为临时跟班，在地图上行窃/抢劫/勒索/间谍。
- **乞丐**：玩家倒闭后原地出现，其他玩家经过须施舍 1000 元。
- **恶犬**：步行玩家碰到后住院 3 天；开车则撞跑恶犬。
- **住院**：丧失行动与收租权利；有保险时可获理赔。
- **坐牢**：丧失行动与收租权利。
- **出国**：无法行动与收租。
