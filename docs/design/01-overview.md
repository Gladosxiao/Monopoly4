# 01. 项目概述

## 项目目标

复刻经典游戏《大富翁4》（Richman 4），打造一个基于 HTML / Web 的多人在线版本，最终部署到 Kimi 的网站。

> 《大富翁4》由大宇资讯（SOFTSTAR）开发，1998 年发行，资料片《超时空之旅》2000 年推出。游戏以掷骰子回合制为核心，支持 1-8 人游玩（本 Web 版聚焦 2-4 人联机）。

## 核心功能

1. **Web 端游戏**：浏览器内可玩的棋盘游戏，支持桌面与移动设备基础适配。
2. **多人在线**：2～4 名玩家实时同局对战，支持创建/加入房间。
3. **登录限制**：仅登录用户可进入房间或开始游戏，未登录用户只能浏览登录页/首页。
4. **经典规则复刻**：
   - 掷骰移动、坐骑系统（步行/机车/汽车）；
   - 土地购买与升级（最高 5 级）；
   - 特殊建筑（公园、商场、旅馆、加油站、研究所）；
   - 连锁店系统；
   - 过路费与连锁加成；
   - 破产判定与胜利条件。
5. **策略卡片系统**：完整 30 张卡片效果已全部落地。
6. **特殊道具系统**：完整 13 种道具效果已全部落地。
7. **神明附身系统**：12 主神附身影响租金、买地/升级费用、住院天数、建筑守护与挡灾。
8. **股市与公司投资**：股票交易（加权成本价/仓位）、持股 >10% 总股本当选董事长、公司地块特效、保险购买与理赔。
9. **事件系统**：命运/新闻事件注册表与效果描述符，支持条件筛选（载具、神明、状态）。
10. **小游戏**：走到小游戏格自动触发七彩气球/喜从天降/企鹅挖宝，根据得分获得点券。
11. **AI 机器人**：测试模式支持添加 AI 机器人补位，自动掷骰/买地/升级。
12. **部署到 Kimi**：构建产物适配 Kimi 网站的托管或运行环境。

## 非目标

- 不追求 100% 还原原游戏美术资源、音效、动画（避免版权风险）。
- 不支持虚拟货币充值、排行榜、社交好友系统等重度社交功能。
- 地图限定事件、更复杂的前端动画与特效待后续细化。

## 目标用户

- 希望通过浏览器重温《大富翁4》的玩家。
- 需要在 Kimi 网站内提供休闲小游戏的场景。
