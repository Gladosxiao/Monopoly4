# 01. 项目概述

## 项目目标

复刻经典游戏《大富翁4》（Richman 4），打造一个基于 HTML / Web 的多人在线版本，最终部署到 Kimi 的网站。

> 《大富翁4》由大宇资讯（SOFTSTAR）开发，1998 年发行，资料片《超时空之旅》2000 年推出。游戏以掷骰子回合制为核心，支持 1-8 人游玩（本 Web 版聚焦 2-4 人联机）。

## 核心功能

1. **Web 端游戏**：浏览器内可玩的棋盘游戏，支持桌面与移动设备基础适配。
2. **多人在线**：2～4 名玩家实时同局对战，支持创建/加入房间。
3. **登录限制**：仅登录用户可进入房间或开始游戏，未登录用户只能浏览登录页/首页。
4. **经典规则复刻**：
   - 掷骰移动、坐骑系统（步行/机车/汽车）；
   - 土地购买与升级（最高 5 级）；
   - 特殊建筑（公园、商场、旅馆、加油站、研究所）；
   - 连锁店系统；
   - 过路费与连锁加成；
   - 破产判定与胜利条件。
5. **策略卡片系统（MVP 核心）**：首期实现 12-15 种基础卡片子集；原版共 30 张卡片，完整列表在设计文档中记录，后续逐步扩展。
6. **特殊道具系统（MVP 核心）**：首期实现交通工具、常见陷阱与工具子集；原版共 13 种道具，完整列表在设计文档中记录，后续逐步扩展。
7. **神明附身系统（扩展）**：财神、福神、衰神、穷神等影响玩家运气，Phase 2+ 实现。
8. **股市与公司投资（扩展）**：实时股票交易、公司持股分红，Phase 2+ 实现。
9. **部署到 Kimi**：构建产物适配 Kimi 网站的托管或运行环境。

## 非目标（首期）

- 不追求 100% 还原原游戏美术资源、音效、动画（避免版权风险）。
- 首期不实现单机 AI 对战（AI 补位作为后续增强）。
- 首期不实现股票系统、公司投资、小游戏、新闻事件、魔法屋、神明附身等复杂子系统（可作为 Phase 2+ 扩展）。
- 首期卡片与道具系统实现基础子集（约 12 种卡片、8 种道具），完整 30+ 种卡片为后续扩展。
- 不支持虚拟货币充值、排行榜、社交好友系统等重度社交功能。

## 目标用户

- 希望通过浏览器重温《大富翁4》的玩家。
- 需要在 Kimi 网站内提供休闲小游戏的场景。
